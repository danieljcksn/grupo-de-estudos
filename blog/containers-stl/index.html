É bastante comum, seja no desenvolvimento ou na resolução de problemas de programação, necessitarmos manipular dados; Pesquisa, busca, inserção e remoção de informações em uma coleção de dados são exemplos de operações recorrentes que todo desenvolvedor utiliza na criação de algoritmos, e para tal existem diversas estruturas de dados: Fila, Lista e Pilha, por exemplo, possuem vasta utilização.

Implementar cada uma dessas estruturas de dados possui custo de tempo elevado, afinal, todas envolvem a utilização de ponteiros, e a construção desses algoritmos exige do programador muita atenção e tempo para que problemas não ocorram. Ter essas estruturas (e outras) implementadas previamente, de modo que sejam facilmente instanciadas ajuda, certamente, na utilização eficaz do tempo e energia do programador. Felizmente, na linguagem C++, isso é uma realidade: STL!

A Standard Template Library (STL) é uma biblioteca padronizada de funções. Esta biblioteca é composta por quatro componentes[1]:

  •    • Contêineres
  •    • Algoritmos
  •    • Iteradores
  •    • Functores


Veremos, neste post, um pouco sobre os contêineres (containers) encontrados em C++.


Contêineres - O que são?

Os contêineres são algoritmos pré-implementados que representam estruturas de dados auxiliares através de classes template. Por serem classes template, tais contêineres podem ser utilizados para armazenar elementos de quaisquer tipos de dados, bem como outras classes[2].
Cada contêiner administra o armazenamento necessário para seus elementos e oferece acesso para cada elemento através de iteradores e/ou funções-membro[3].

Existem 4 tipos de contêiner em C++, são eles:

  •    1. Sequenciais
  •    2. Adaptativos
  •    3. Associativos
  •    4. Associativos Desordenados

Vejamos um pouco sobre cada um desses contêineres.


1. Contêineres Sequenciais

Os contêineres sequenciais são representações de estruturas de dados que armazenam elementos homogêneos, de modo que cada elemento do contêiner pode ser acessado sequencialmente.
Os contêineres sequenciais presentes na atual versao do C++ são array, vector, deque, forward_list e list.


1.1 Array

O contêiner array representa um vetor estático e contínuo (os elementos são alocados sequencialmente na memória, mas o número máximo de elementos não pode ser alterado). Esse contêiner foi criado com a finalidade de ser uma releitura do array presente na linguagem C, porém com a mesma performance e maior legibilidade.
A interface do array presente na STL torna mais conveniente as operações para encontrar o tamanho do array e copiar elementos[4], por exemplo, bem como permite a utilização de algoritmos baseados em iteradores, definidos na STL (falaremos sobre iteradores no próximo post).

Funções Membro de std::array

at

Retorna o valor no array da posição informada. Se a posição dada for maior que o tamanho do array, a exceção out_of_range será exibida.

[ ]

Assim como o acesso à um elemento do vetor padrão, os elementos do array podem ser acessados diretamente através do operador [].

front( )

Retorna o primeiro elemento do array.

back( )

Retorna o último elemento do array.

fill( )

Preenche todos os espaços do array com o valor informado.

swap( )

Preenche todos os espaços do array com o valor informado.

size( )

Retorna o tamanho do array.

empty( )

Verifica se o array está vazio. Retorna true se estiver vazio e false caso contrário.
